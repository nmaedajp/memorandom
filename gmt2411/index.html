GMTの図の貼り込み

x = rand(1000)*2
y = randn(1000)
flout = joinpath(@OUTPUT, "test1")
using GMT
gmtbegin(flout, fmt="png")
   GMT.scatter(x, y, region=(-0.1,2.1,-4,4), J="X6c/6c", B="afg")
gmtend()

// Image matching '/assets/gmt2411/output/test1' not found. //

  • gmtbeginで図をファイル出力し，出力を貼り込む．

  • @OUTPUT を使って出力先を指定する．

    • __site/assets の中の適当な場所を指定できるようだ．
